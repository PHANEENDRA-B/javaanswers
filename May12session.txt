When we declare the class, is any constructor added automatically by the compiler?
Yes, the default constructor (without any arguments) will get added only if no constructors

What is a constructor? What is its role in a class?
Constructor is a special method which has the same name as the class. 
This method gets invoked immediately after allocation of memory using new keyword.
It is used to initialize the state of the object (provide values to the data members).

What is the access modifier of the class and its data members in the above code?
If we do not mention any access modifier in Java, the default access modifier is package private.
It means we can access the data members within the class domain and from any other class in the same package.

a) the method overridden in the child class must have the same name and signature 
as the method inherited from the base class.

SUB TYPE of the return type of the method inherited from the base class.

The reference variable of super type can be used to assign the address of the object of type child.
When you use such a reference to invoke a method which is present in both base and child class,

variable. Hence, in the above context, the accelerate() method defined in the class Car will be executed.

A class can extend only one single class in Java.
A class can implement multiple interfaces.
We define the interface as a contract which outlines the services to be provided by the interface.
The actual implementation is to be given by the concrete class implementing the interface.

A class first extends then implements.

Till Java7, an interface could only have public abstract methods.
From Java8, an interface can have both abstract, non abstract as well as static methods.

An interface with one and only one abstract method is better known as Functional Interface
An interface without any method is known as MARKER interface like java.io.Serialiozable interface.
Marker Interface (no methods or fields)

The static method defined in an interface can only be accessed using interface name.

class Demo{
 	public static void main(String[] a){
		Vehicle vehicle=new Car();
		vehicle.accelerate();
		Vehicle.checkWheels();	//should be accessed using interface name
		vehicle.drive();

class Car implements Vehicle{	
	public void accelerate(){
		System.out.println("The car can accelerate plus apply brakes");
	}

interface Vehicle{	
	void accelerate();// access modifier is public 

	static void checkWheels(){
		System.out.println("All wheels doing good");
	}

	default void drive(){
		System.out.println("Right hand driving in India");
	}	
}
